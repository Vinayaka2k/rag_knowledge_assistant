Password Reset Guide

If a user cannot log into their account due to a forgotten password:

1. Direct the user to the password reset page.
2. The user should enter their registered email address.
3. A reset link will be sent to the email.
4. The link expires after 30 minutes.
5. If the user does not receive the email, verify the spam folder.

Security Note:
Never manually set passwords for users unless authorized by the security team.
